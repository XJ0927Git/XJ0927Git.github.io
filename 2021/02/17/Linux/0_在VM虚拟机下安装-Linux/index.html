安装Linux

1） Create a New Virtual Machine (新建虚拟机)

2）选择Typical(典型)

3）稍后选择 Linux CentOS安装镜像

4）选择CentOS安装目录

5）给系统指定最大磁盘空间(不建议小于5G)

6）完成

7）开始Server安装

8）开启虚拟机

若第一次安装,此时可能出现下列问题

解决措施:

14）设置root账户

17）设置时间

18）登录

19）网卡设置(非必须)

nat方式，每次换网络的情况下，IP不会变化。跨网段访问。
桥接模式，每次换网络的情况下，IP都会变化。优势是和主机一个段，相对更快。
